# Angular 17 with Material 3 (experimental)

## About this project

This project is about implementing the usage of the brand-new Material 3 UI
library with Angular 17.

## Requirements

| Name        | Version |
| ----------- | ------- |
| Node.js     | 20.11.1 |
| npm         | 10.2.4  |
| Angular CLI | 17.2.2  |
| TypeScript  | 5.3.2   |
